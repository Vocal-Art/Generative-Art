DEFINE GENERETIF ART


Generative art is a form of art produced by an autonomous system rather than being created directly by the artist.

The system can be a computer program, but it can also be biological, social, or other types of systems.

The artist may be involved in defining certain parameters that influence the final outcome, but the result is generated by the system!

Algorithmic art uses algorithms implemented as computer programs.

DESCRIPTION OF MY WORK

For this project, I followed the instructions of the original artists and reproduced the codes line by line.

I modified parameters in the code to create unique variations in the generated works.

I invested a lot of time and effort to study each piece, gradually learning programming concepts in a self-taught manner.

Therefore, this project is a digital and visual reproduction of works created by other artists, using modified parameters from the original pieces.

The goal is to create personal and random generative works while drawing inspiration from the original creations.
